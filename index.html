Pupular HTML Tags

S. No. Tag Description Code snippet
1 <HTML> This tag is use to initise or tell the browser that it is a html page. <HTML>This is an html page.</HTML>
2 <HEAD> This tag is use to specify table address fields like title, meta data or some script that need to be loaded before body get loaded. <HTML>< HEAD> <TITLE>This is title in header </TITLE> </HEAD ></HTML>
3 <TITLE> This tag is used to define the title of the html web page and it is also use for search engine to indentify the main content in the page. <HTML>< HEAD> <TITLE>This is title in header </TITLE> </HEAD ></HTML>
4 <META> This tag is used to data or description about the page like author, description, keywords, and other data about the page. <HTML>< HEAD> <META NAME="author" CONTENT="Vish Sharma" > <META NAME="keywords" CONTENT="top, html, tags, tutorial" > </HEAD ></HTML>
5 <SCRIPT> This tag is used to add any scripting langauge code to page. Generally we use it for JavaScript or JQuery. <HTML>< HEAD> <SCRIPT>console.log("Hello Everyone!")</SCRIPT> <SCRIPT SRC="app.js"></SCRIPT> </HEAD ></HTML>
6 <STYLE> This tag is use to add styling to the page by the help of CSS. <HTML>< HEAD> <STYLE>#title { color: black; }</STYLE> </HEAD ></HTML>
7 <BODY> Whatever we write in body tag is only visible in the HTML page content. <HTML> <BODY> <P>This is a paragraph inside body tag.</P> </BODY> </HTML>
8 <IMAGE> This tag is use to display image in HTML page. <HTML> <BODY> <IMAGE SRC="img1.jpg" HEIGHT="100" WIDTH="200"></IMAGE> </BODY> </HTML>
9 <FORM> This tag is use to create an form in html page so that use input can be taken. <HTML> <BODY> <FORM METHOD="POST" ACTION="insert.php"></FORM> </BODY> </HTML>
10 <INPUT> This tag is use to get input from user by various method like text, image, number, date, time etc. <HTML> <BODY> <FORM METHOD="POST" ACTION="insert.php"> <INPUT TYPE="text"> <INPUT TYPE="file"> </FORM> </BODY> </HTML>
11 <TEXTAREA> This tag is use to get multiline text as input from user. <HTML> <BODY> <FORM METHOD="POST" ACTION="insert.php"> <TEXTAREA> </TEXTAREA> </FORM> </BODY> </HTML>
12 <LI> This tag is use to display ordered(<ol>) and unorder(<ul>) lists. <HTML> <BODY> <LI> <UL>item 1</UL> <UL>item 2</UL> </LI> <LI> <OL>item 1</OL> <OL>item 2</OL> </LI> </BODY> </HTML>
13 <TABLE> This tag is use to display table in html page. <HTML> <BODY> <TABLE> <TR> <TD> this is table data. </TD> </TR> </TABLE> </BODY> </HTML>
14 <A> This tag is use to add hyperlink in the page to connect multiple pages. <HTML> <BODY> <A HREF="page1.html"> Click here... </A> </BODY> </HTML>
15 <P> This tag is use to display paragraph in HTML page. <HTML> <BODY> <P> This is a paragraph. </P> </BODY> </HTML>
16 <H1> This tag is use to display heading in HTML page(H1-H7). <HTML> <BODY> <H1> This is a largest heading. </H1> </BODY> </HTML>
17 <HEADER> This tag is use to add header content in html page. <HTML> <BODY> <HEADER> Header will goes here. </HEADER> </BODY> </HTML>
18 <FOOTER> This tag is use to add footer content in html page. <HTML> <BODY> <FOOTER> footer will goes here. </FOOTER> </BODY> </HTML>
19 <SECTION> This tag is use to add section content in html page. <HTML> <BODY> <SECTION> This is a new section. </SECTION> </BODY> </HTML>
20 <MAIN> This tag is use to add main content in html page. <HTML> <BODY> <MAIN> This is main section. </MAIN> </BODY> </HTML>
